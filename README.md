# pxm-vue

`基于vue-cli 4.5.0搭建的SPA模版`

# 技术栈

`vue + ts + rxjs`

# 项目描述

`项目在utils文件夹下封装了部分函数可供使用，后续也会陆续增加`

`项目引入TypeScript来规范代码书写`

`项目引入RxJS响应式编程库，可以编写复杂应用逻辑`

`初始化项目已mock好了Authentication、StremFlow两个类，用于帮助快速熟悉rxjs的使用（如果你rxjs已经非常熟悉，请忽略～）`
